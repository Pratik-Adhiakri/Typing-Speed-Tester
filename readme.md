# How does it work?

Type Rush works on a very simple logic when YOU type a word and complete it then the code first receives that and then checks if it is right or not and depending if it is right or not it increases/decreases accuracy and then this process repeats then the code ends the game acc to the time then after the game finishes then immediately after, the code receives all the words typed in the specific time and checks accuracy then it multiplies both then the final wpm comes. So, for example if you typed 60 words with 100% accuracy in 30 secs then it will be 60 * 1.0 *(60/30). Then the result will be 120 wpm! I hope you understood!
